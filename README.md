# Version Control Skills Lab

## What this project does
This project contains a Jupyter notebook program that asks the user for their name and a whole number (1–20).
The computer then prints a personalized “power level” bar and repeats a custom message the number of times entered.

## How to run
Open `main.ipynb` in VS Code (Jupyter extension) or JupyterLab and run the cells.
